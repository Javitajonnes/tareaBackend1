🎨 Preview de Fuentes para tu Sitio Web

1. Inter

Título Principal (Weight 600)

Este es un párrafo normal con peso 400. Inter es una fuente moderna y muy legible, perfecta para sitios web profesionales.

Texto más ligero con peso 300. Muy elegante para subtítulos y elementos secundarios.

font-family: 'Inter', sans-serif;
💡 Recomendación: Muy moderna, excelente legibilidad, perfecta para sitios profesionales.
